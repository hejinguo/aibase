为规范并预防项目组内成员随意引用各自的库导致版本或者实现效果的差异，
这里约定系统可引用插件库的说明如下(若业务需要想引入其他三方库请告知项目成员并进行评估后纳入本文)：
------------------------------------------------------------
1、核心库(每个页面均需要引入的库)
jquery			jquery库
bootstrap 		前端开源工具包
html5shiv 		Bootstrap兼容IE8插件
respond 		Bootstrap兼容IE8插件
art-template 	JS模板引擎
------------------------------------------------------------
2、根据业务需要引入的库
moment 			日期处理库
typeahead 		基于Bootstrap的自动完成
powerFloat 		万能浮动层插件
slimScroll 		滚动条插件
echarts 		百度图表库
ueditor 		百度编辑器
ztree 			树形结构
------------------------------------------------------------
3、业务需要可替换Bootstrap中组件使用的库，但需保证UI风格一致
layer 			弹窗插件，也可直接使用Bootstrap提供的弹窗
laypage 		分页插件，也可直接使用Bootstrap提供的分页
